USENIX Security 2014 Acceptance Rate

USENIX Security 2014 accepted 67 papers out of 350 submissions, for an acceptance rate of 19.1%.

USENIX Security Symposium. 2014 (23rd) was held in San Diego, California, USA.

Compared with USENIX Security 2013, the acceptance rate increased from 15.9% to 19.1%, and submissions increased from 277 to 350.
