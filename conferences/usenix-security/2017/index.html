USENIX Security 2017 Acceptance Rate

USENIX Security 2017 accepted 85 papers out of 553 submissions, for an acceptance rate of 15.4%.

USENIX Security Symposium. 2017 (26th) was held in Vancouver, Canada.

Compared with USENIX Security 2016, the acceptance rate decreased from 16.0% to 15.4%, and submissions increased from 451 to 553.
